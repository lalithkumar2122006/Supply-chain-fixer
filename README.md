# 🚚 Supply Chain Fixer

An interactive web application that simulates a supply chain and demonstrates how disruptions can be detected and resolved using a multi-agent workflow. The project visualizes the movement of products from suppliers to customers while automatically responding to operational issues such as delays, low inventory, and transportation failures.

---

## 📌 Project Overview

Supply Chain Fixer is a simulation platform designed to help users understand how a supply chain operates and how disruptions affect the overall workflow.

The application models four major stages of a supply chain:

- 🏭 Supplier
- ⚙️ Manufacturer
- 🚛 Distributor
- 🏪 Retailer

When disruptions occur, dedicated agents detect the issue, analyze the impact, and coordinate recovery actions to restore normal operations.

---

## 🤖 Multi-Agent System

The project includes specialized agents that work together:

### Supplier Agent
- Manages raw material availability
- Detects supplier delays

### Manufacturer Agent
- Monitors production
- Handles machine breakdown scenarios

### Distributor Agent
- Tracks shipments
- Detects transportation delays

### Retailer Agent
- Monitors stock levels
- Detects inventory shortages

### Fixer Agent
- Collects alerts from all agents
- Identifies the root cause
- Executes recovery strategies
- Restores smooth supply chain operations
